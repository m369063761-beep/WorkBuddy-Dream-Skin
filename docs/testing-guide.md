# V0.2.2 人工验收指南

这份清单用于宣传前的首次用户验收。建议准备一张小于 12 MB 的 JPG 或 PNG 横图，并在测试过程中避免让 WorkBuddy 执行重要任务。

## 1. 从零安装

1. 下载 `WorkBuddy-Dream-Skin-Windows-v0.2.2.zip`，解压到任意普通文件夹。
2. 双击 `Install WorkBuddy Dream Skin.cmd`。
3. 确认主题中心自动打开。
4. 检查桌面和开始菜单是否出现 `WorkBuddy Dream Skin` 快捷方式。

预期：不要求管理员权限；界面显示至少 6 个主题，包括 Dream Glass、Sakura Night、路飞五档、草帽团、守岸人和长离配色壳。

## 2. 内置主题

1. 从 WorkBuddy 托盘菜单完全退出 WorkBuddy。
2. 在主题中心选择 `Dream Glass`，点击“应用主题”。
3. 等待 WorkBuddy 启动。
4. 返回主题中心，选择 `Sakura Night`，再次点击“应用主题”。

预期：WorkBuddy 首次自动启动；第二次无需重启即可切换；背景、透明面板和圆角立即变化。

## 3. 自定义图片

1. 选择任意主题作为基础样式。
2. 点击“导入自己的图片”，选择准备好的 JPG、PNG、WebP 或 GIF。
3. 确认列表出现以“我的图片 ·”开头的新主题。
4. 点击“应用主题”。
5. 关闭主题中心，再通过桌面快捷方式重新打开。

预期：预览区显示所选图片；WorkBuddy 使用这张图片；重新打开后自定义主题仍存在。图片只应位于 `%LOCALAPPDATA%\WorkBuddyDreamSkin\themes-local`。

## 4. 首页与任务页

1. 在 WorkBuddy 首页观察主题背景。
2. 打开一个普通任务页。
3. 返回首页。

预期：首页更突出背景；任务页自动加深蒙层，保证文字可读；来回切换不需要重新应用主题。

## 5. 恢复与再次应用

1. 在主题中心点击“恢复官方外观”。
2. 确认背景和透明样式消失。
3. 再选择任意主题并点击“应用主题”。

预期：恢复操作不会删除图片或主题；之后可以再次应用。

## 6. 异常提示

1. 恢复官方外观，然后完全关闭 WorkBuddy。
2. 用官方快捷方式正常启动 WorkBuddy。
3. 在主题中心点击“应用主题”。

预期：工具不会强制关闭正在运行的 WorkBuddy，而是提示从托盘完全退出；当前任务不会被工具主动中断。

## 7. 客户定制包

1. 双击 `制作客户定制包.cmd`。
2. 填写测试客户名称，选择一张本地照片和基础配色。
   默认保持“自动匹配照片（推荐）”，确认界面显示识别出的浅色/深色模式和主色值。
3. 生成 ZIP，并确认同时出现 `.sha256` 文件。
4. 解压 ZIP，确认存在 `客户安装说明.txt` 和客户专属主题目录。
5. 在另一测试安装目录运行安装器，确认主题中心自动选中客户主题。

预期：生成包不包含卖方的客户包生成器；照片只存在于客户 ZIP 内；客户不需要 Git 或管理员权限。

## 8. 卸载

1. 从开始菜单打开 `卸载 WorkBuddy Dream Skin`。
2. 等待几秒。
3. 检查桌面快捷方式、开始菜单目录和 `%LOCALAPPDATA%\WorkBuddyDreamSkin`。

预期：快捷方式和安装目录被删除；腾讯 WorkBuddy 本身仍然存在且可正常启动。
